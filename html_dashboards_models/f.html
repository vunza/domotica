Renomear Dispositivo

Para renomear o dispositivo, acesse a interface de configuração via API REST.

Atenção: Esta interface é apenas para visualização. Para aplicar as alterações, use a API do dispositivo.

Atualização OTA

Abaixo está a interface do AsyncElegantOTA para upload de firmware.

Instruções: O dispositivo deve estar no modo OTA. Acesse a página padrão do ElegantOTA em /update para fazer o upload do firmware.

Para iniciar o modo OTA no dispositivo, envie uma requisição POST para /api/ota
